Headings and Text

Headings and paragraphs are the foundation of readable content. They help organize information clearly on a webpage.

What You Will Learn

  • How to use heading tags
  • The difference between h1 to h6
  • How to write paragraphs

HTML Headings

HTML provides six levels of headings:

<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection</h3>
<h4>Smaller Heading</h4>
<h5>Even Smaller</h5>
<h6>Smallest Heading</h6>

Use headings to structure your content. Think of them like chapters and sections in a book.

Paragraphs

<p>This is a paragraph.</p>

The <p> tag is used to write blocks of text. It helps keep your content readable and organized.

Best Practice

  • Use only one <h1> per page
  • Follow a logical order (h1 → h2 → h3)
  • Use paragraphs to separate ideas

Try It Yourself

Create a simple page:

  • Add an h1 with your name
  • Add an h2 with a section title
  • Write a paragraph about yourself
