HTML Basics, Your First Web Page

This is where everything begins. HTML is the foundation of every website. Let’s build your first webpage step by step.

What is HTML?

HTML (HyperText Markup Language) is used to structure content on the web. It tells the browser what each part of a page is—like headings, paragraphs, links, and images.

What You Will Learn

  • Basic structure of an HTML page
  • How to create headings and paragraphs
  • How a webpage is organized

Your First HTML Page

Here is a simple example:

<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>

  <h1>Hello World</h1>
  <p>This is my first webpage.</p>

</body>
</html>

Save this as index.html and open it in your browser.

Understanding the Structure

  • <!DOCTYPE html> defines the document type
  • <html> root element
  • <head> contains metadata
  • <body> visible content

Try It Yourself

Create your own HTML page:

  • Add your name as a heading
  • Write a short paragraph about yourself
  • Change the title of the page
