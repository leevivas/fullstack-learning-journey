Links and Images

Links connect pages, and images make your website more engaging. These are essential building blocks of any webpage.

What You Will Learn

  • How to create links using the anchor tag
  • How to add images to a webpage
  • How to use attributes like href, src, and alt

Creating Links

<a href="https://www.google.com">Visit Google</a>

The <a> tag is used to create links. The href attribute defines the destination.

Adding Images

<img src="image.jpg" alt="Description of image">

The <img> tag is used to display images. The src attribute specifies the image path, and alt provides a description.

Best Practice

  • Always use meaningful link text
  • Always include alt text for images
  • Use relative paths for internal links

  • Add an image to your page
