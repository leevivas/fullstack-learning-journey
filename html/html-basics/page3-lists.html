Lists

Lists help organize information clearly. They are useful for steps, items, and grouped content on a webpage.

What You Will Learn

  • How to create unordered lists
  • How to create ordered lists
  • How to nest lists

Unordered List

<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Orange</li>
</ul>

The <ul> tag creates a bulleted list.

Ordered List

<ol>
  <li>Wake up</li>
  <li>Study HTML</li>
  <li>Build a webpage</li>
</ol>

The <ol> tag creates a numbered list.

Nested Lists

<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
</ul>

Lists can be nested to show hierarchy.

Best Practice

  • Use lists to organize related content
  • Keep list items short and clear
  • Use ordered lists for steps

Try It Yourself

  • Create a list of your favorite foods
  • Create a step-by-step routine using ordered list
  • Create a nested list of programming skills
