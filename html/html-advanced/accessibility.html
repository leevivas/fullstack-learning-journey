Accessibility

Accessibility means making your website usable for everyone, including people with disabilities.

What You Will Learn

  • What accessibility means
  • How to use alt text for images
  • How semantic HTML improves accessibility

Why Accessibility Matters

Not all users interact with websites the same way. Some use screen readers, keyboard navigation, or assistive technologies.

Accessible websites ensure that everyone can understand and use your content.

Using Alt Text

<img src="image.jpg" alt="A description of the image">

The alt attribute describes images for users who cannot see them.

Labels for Forms

<label for="email">Email:</label>
<input id="email" type="email">

Labels help users understand what each input field is for.

Use Semantic HTML

Using tags like <header>, <nav>, and <main> helps screen readers understand your page.

Best Practice

  • Always include alt text for images
  • Use proper labels for forms
  • Use semantic HTML elements
  • Ensure good contrast and readability

Try It Yourself

  • Add alt text to all images on your page
  • Create a form with labels
  • Use semantic tags in your layout
