Tables

Tables are used to display structured data in rows and columns. They are useful for organizing information clearly.

What You Will Learn

  • How to create a basic table
  • How to use rows and columns
  • How to use table headers

Basic Table Example

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>20</td>
  </tr>
</table>

This creates a simple table with two columns.

Table Structure

  • <table> defines the table
  • <tr> table row
  • <th> header cell
  • <td> data cell

Best Practice

  • Use tables only for data, not layout
  • Always use headers for clarity
  • Keep tables simple and readable

Try It Yourself

  • Create a table with 3 columns (Name, Course, Year)
  • Add at least 3 rows of data
  • Use table headers
