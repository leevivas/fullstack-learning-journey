Forms

Forms allow users to send data to a website. They are used for login pages, registration, search, and more.

What You Will Learn

  • How to create a basic form
  • How to use input fields
  • How to add buttons and labels

Basic Form Example

<form>
  <label>Name:</label>
  <input type="text">
  <br>
  <button>Submit</button>
</form>

This creates a simple form with a text input and a submit button.

Common Input Types

<input type="text">
<input type="password">
<input type="email">
<input type="checkbox">
<input type="radio">

Different input types allow you to collect different kinds of data.

Using Labels

<label for="name">Name:</label>
<input id="name" type="text">

Labels improve usability and accessibility by connecting text to inputs.

Best Practice

  • Always use labels for inputs
  • Use appropriate input types
  • Keep forms simple and clear

Try It Yourself

  • Create a login form (username + password)
  • Add a submit button
  • Try different input types
