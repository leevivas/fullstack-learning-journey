Loops

Loops allow you to repeat a block of code multiple times. This helps automate tasks and reduce repetitive code.

What You Will Learn

  • How to use for loops
  • How to use while loops
  • How to repeat tasks efficiently

For Loop

for (let i = 0; i < 5; i++) {
  console.log(i);
}

The for loop is used when you know how many times to repeat.

While Loop

let i = 0;

while (i < 5) {
  console.log(i);
  i++;
}

The while loop runs as long as the condition is true.

Do...While Loop

let i = 0;

do {
  console.log(i);
  i++;
} while (i < 5);

This loop runs at least once before checking the condition.

Looping Through Arrays

let fruits = ["apple", "banana", "orange"];

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}

Loops are commonly used to process lists of data.

Best Practice

  • Use for loops for counting
  • Use while loops for conditions
  • Avoid infinite loops

Try It Yourself

  • Print numbers from 1 to 10
  • Loop through an array of names
  • Create a loop that counts down
