Conditions

Conditions allow your program to make decisions. Your code can respond differently depending on different situations.

What You Will Learn

  • How to use if statements
  • How to use else and else if
  • How to use switch statements

If Statement

let age = 18;

if (age >= 18) {
  console.log("You are an adult");
}

The if statement runs code only if the condition is true.

If...Else

let age = 16;

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

else runs when the condition is false.

Else If

let score = 85;

if (score >= 90) {
  console.log("Excellent");
} else if (score >= 75) {
  console.log("Good");
} else {
  console.log("Needs Improvement");
}

Use else if to check multiple conditions.

Switch Statement

let day = 1;

switch(day) {
  case 1:
    console.log("Monday");
    break;
  case 2:
    console.log("Tuesday");
    break;
  default:
    console.log("Other day");
}

switch is useful when checking multiple fixed values.

Comparison Operators

  • == equal
  • === strictly equal
  • != not equal
  • > greater than
  • < less than

Best Practice

  • Use === instead of ==
  • Keep conditions simple
  • Use switch for multiple fixed values
