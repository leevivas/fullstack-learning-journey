JavaScript Basics

JavaScript allows you to add logic and interactivity to your website. Let’s start with the fundamentals.

What You Will Learn

  • How to write JavaScript code
  • Variables and data types
  • Basic syntax and structure

How to Use JavaScript

<script>
  console.log("Hello, World!");
</script>

JavaScript can be written inside the <script> tag or in a separate file.

Variables

let name = "John";
const age = 20;
var city = "Manila";

Variables store data. Use let and const in modern JavaScript.

Data Types

let text = "Hello";      // String
let number = 10;         // Number
let isActive = true;     // Boolean
let items = [1, 2, 3];   // Array
let person = { name: "Ana" }; // Object

JavaScript supports different types of data for different use cases.

Basic Syntax

let x = 5;
let y = 10;

let sum = x + y;

console.log(sum);

JavaScript uses statements and expressions to perform operations.

Best Practice

  • Use const for values that don’t change
  • Use let for variables that change
  • Avoid using var

Try It Yourself

  • Create a variable with your name
  • Create a variable with your age
  • Print them using console.log()
