Functions

Functions allow you to group code into reusable blocks. They help you organize your logic and avoid repetition.

What You Will Learn

  • What functions are
  • How to create and use functions
  • How to pass data using parameters

Basic Function

function greet() {
  console.log("Hello!");
}

greet();

A function runs when it is called.

Parameters

function greet(name) {
  console.log("Hello " + name);
}

greet("John");

Parameters allow you to pass values into a function.

Return Values

function add(a, b) {
  return a + b;
}

let result = add(5, 3);
console.log(result);

return sends a value back from a function.

Arrow Functions (Modern JS)

const greet = (name) => {
  console.log("Hello " + name);
};

Arrow functions are a shorter way to write functions.

Best Practice

  • Use clear and meaningful names
  • Reuse functions instead of repeating code
