Modern JavaScript (ES6+)

Modern JavaScript introduces powerful features that make code cleaner, shorter, and easier to manage.

What You Will Learn

  • Arrow functions
  • Destructuring
  • Modules

Arrow Functions

const greet = (name) => {
  return "Hello " + name;
};

Arrow functions are a shorter way to write functions.

Destructuring

let person = {
  name: "Ana",
  age: 20
};

let { name, age } = person;

console.log(name);

Destructuring allows you to extract values from objects easily.

Array Destructuring

let colors = ["red", "blue", "green"];

let [first, second] = colors;

console.log(first);

Modules

// file1.js
export const name = "John";

// file2.js
import { name } from "./file1.js";

Modules allow you to split code into separate files.

Template Literals

let name = "John";
console.log(`Hello ${name}`);

Use backticks (`) for easier string formatting.

Best Practice

  • Use arrow functions for short functions
  • Use destructuring for cleaner code
  • Organize code using modules

Try It Yourself

  • Rewrite a function using arrow syntax
  • Extract values using destructuring
  • Create two JS files using modules
