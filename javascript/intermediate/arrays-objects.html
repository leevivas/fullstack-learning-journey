Arrays & Objects

Arrays and objects allow you to store and organize complex data. They are essential for building real applications.

What You Will Learn

  • How to create arrays
  • How to create objects
  • How to access and modify data

Arrays

let fruits = ["apple", "banana", "orange"];

Arrays store multiple values in a list. 

console.log(fruits[0]); // apple

Array Methods

fruits.push("mango");   // add item
fruits.pop();           // remove last item

Arrays have built-in methods to manage data.

Objects

let person = {
  name: "John",
  age: 25
};

Objects store data in key-value pairs. 

console.log(person.name);

Modify Data

person.age = 30;
fruits[1] = "grape";

You can update values inside arrays and objects.

Array of Objects

let users = [
  { name: "Ana", age: 20 },
  { name: "Leo", age: 22 }
];

console.log(users[0].name);

This structure is commonly used in real applications.

Best Practice

  • Use arrays for lists
  • Use objects for structured data
  • Keep data organized and readable

Try It Yourself

  • Create an array of your favorite foods
  • Create an object representing yourself
  • Create an array of objects (students, products, etc.)
