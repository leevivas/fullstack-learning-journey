DOM Selectors

The DOM (Document Object Model) allows JavaScript to access and manipulate HTML elements.

What You Will Learn

  • What the DOM is
  • How to select elements using JavaScript
  • Different selector methods

What is the DOM?

The DOM represents your HTML page as a structure that JavaScript can interact with. It allows you to access and modify elements dynamically.

Selecting Elements

let title = document.getElementById("title");

This selects an element by its ID.

Query Selector

let element = document.querySelector(".className");

querySelector allows you to use CSS-style selectors.

Select Multiple Elements

let items = document.querySelectorAll("p");

This selects all matching elements.
