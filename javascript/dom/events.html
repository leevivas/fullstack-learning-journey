Events

Events allow your website to respond to user actions such as clicks, typing, and mouse movements.

What You Will Learn

  • What events are
  • How to handle user actions
  • Common event types

What is an Event?

An event is something that happens in the browser, like a click, a key press, or a page load.

Click Event

button.addEventListener("click", function() {
  console.log("Button clicked!");
});

Input Event

input.addEventListener("input", function() {
  console.log(input.value);
});

Common Events

  • click
  • input
  • change
  • submit
  • mouseover

Best Practice

  • Use addEventListener instead of inline events
  • Keep event functions simple
  • Avoid repeating code

Try It Yourself

  • Create a button that shows an alert
  • Display input text in real time
  • Change background color on click
