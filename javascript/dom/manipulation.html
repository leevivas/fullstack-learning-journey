DOM Manipulation

DOM manipulation allows you to change the content, style, and structure of your webpage dynamically using JavaScript.

What You Will Learn

  • How to change text content
  • How to modify styles
  • How to create and remove elements

Change Content

let el = document.getElementById("text");
el.textContent = "New Text!";

Original Text

Change Style

el.style.color = "red";

Watch my color change

Create Element

let newEl = document.createElement("p");
newEl.textContent = "New paragraph";
document.body.appendChild(newEl);

Remove Element

el.remove();

Remove me

Best Practice

  • Separate JavaScript from HTML when possible
  • Reuse selected elements
  • Avoid excessive DOM updates

Try It Yourself

  • Create a button that changes text
  • Change styles dynamically
  • Add and remove elements
