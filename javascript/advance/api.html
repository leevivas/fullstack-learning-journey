Working with APIs (Fetch)

APIs allow your application to communicate with external services. You can fetch real data and display it on your website.

What You Will Learn

  • What an API is
  • How to use fetch()
  • How to display data dynamically

What is an API?

An API (Application Programming Interface) allows your app to request data from a server.

Basic Fetch Example

fetch("https://jsonplaceholder.typicode.com/posts/1")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.log(error));

Using Async/Await

async function getPost() {
  try {
    let response = await fetch("https://jsonplaceholder.typicode.com/posts/1");
    let data = await response.json();
    console.log(data);
  } catch (error) {
    console.log(error);
  }
}

getPost();

Display Data on Page

Best Practice

  • Always handle errors
  • Use async/await for cleaner code
  • Validate API responses
