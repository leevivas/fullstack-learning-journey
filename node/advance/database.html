Database Integration

Databases allow your application to store and manage data. This is essential for building real-world applications.

What You Will Learn

  • What a database is
  • How to connect Node.js to a database
  • Basic CRUD operations

What is a Database?

A database stores information such as users, products, and tasks. Your application can create, read, update, and delete data.

Types of Databases

  • NoSQL — MongoDB
  • SQL — MySQL, PostgreSQL

Example: MongoDB with Mongoose

npm install mongoose
      
const mongoose = require("mongoose");

mongoose.connect("mongodb://localhost:27017/myapp")
  .then(() => console.log("Connected to DB"))
  .catch(err => console.log(err));

Create a Model

const User = mongoose.model("User", {
  name: String,
  age: Number
});

Create Data

const user = new User({ name: "John", age: 25 });
user.save();

Read Data

User.find().then(users => console.log(users));

Update Data

User.updateOne({ name: "John" }, { age: 30 });

Delete Data

User.deleteOne({ name: "John" });

CRUD Operations

  • Create
  • Read
  • Update
  • Delete

Best Practice

  • Validate data before saving
  • Handle database errors
  • Separate database logic into modules
