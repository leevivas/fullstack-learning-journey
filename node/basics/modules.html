Modules

Modules allow you to split your code into smaller, reusable files. This makes your application easier to manage and scale.

What You Will Learn

  • What modules are
  • How to export code
  • How to import modules

What is a Module?

A module is a separate file that contains code you can reuse in other parts of your application.

Exporting Code

// math.js
function add(a, b) {
  return a + b;
}

module.exports = add;

Use module.exports to share code.

Importing Code

// app.js
const add = require("./math");

console.log(add(2, 3));

Use require() to import modules.

Export Multiple Functions

// math.js
module.exports = {
  add,
  subtract
};

You can export multiple functions as an object.

Built-in Modules

  • http → create servers
  • fs → file system
  • path → file paths

Best Practice

  • Keep modules small and focused
  • Group related functions
  • Use clear file names

Try It Yourself

  • Create a math module
  • Export multiple functions
  • Import and use them in another file
