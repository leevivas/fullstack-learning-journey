Node.js Basics

Node.js allows you to run JavaScript on the server. Let’s build your first backend application.

What You Will Learn

  • What Node.js is
  • How to run JavaScript outside the browser
  • How to create a simple server

What is Node.js?

Node.js is a runtime environment that allows JavaScript to run on your computer or server instead of just inside a browser.

Run JavaScript with Node

node app.js

This command runs your JavaScript file using Node.js.

Create Your First Server

const http = require("http");

const server = http.createServer((req, res) => {
  res.write("Hello from Node.js!");
  res.end();
});

server.listen(3000, () => {
  console.log("Server running on http://localhost:3000");
});

This creates a basic server that responds with text.

How It Works

  • http → built-in module
  • createServer() → creates server
  • req → request from user
  • res → response sent back

Test Your Server

Open your browser and go to: 

http://localhost:3000

You should see your message.

Best Practice

  • Keep server code simple
  • Use modules to organize code
  • Always handle requests properly

Try It Yourself

  • Change the port number
