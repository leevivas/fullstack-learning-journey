Express.js

Express.js is a framework for Node.js that simplifies building web servers and APIs. It is widely used in real-world applications.

What You Will Learn

  • What Express.js is
  • How to create a server using Express
  • How to build API endpoints

What is Express.js?

Express.js is a minimal and flexible Node.js framework that makes it easier to build servers and handle requests.

Install Express

npm init -y
npm install express

These commands set up your project and install Express.

Create a Basic Server

const express = require("express");
const app = express();

app.get("/", (req, res) => {
  res.send("Hello from Express!");
});

app.listen(3000, () => {
  console.log("Server running on http://localhost:3000");
});

This creates a simple server with one route.

Routes (Endpoints)

app.get("/about", (req, res) => {
  res.send("About Page");
});

app.get("/api/data", (req, res) => {
  res.json({ message: "Hello API" });
});

Routes define how your server responds to requests.

Send JSON Data

res.json({ name: "John", age: 25 });

APIs often send data in JSON format.

Why Express Matters

  • Build REST APIs
  • Handle HTTP requests easily
  • Used in production applications

Best Practice

  • Keep routes organized
  • Use middleware for logic
  • Separate files using modules
