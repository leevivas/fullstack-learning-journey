Build a Backend API

In this project, you will build a real API using Node.js and Express. This is how modern web applications communicate with servers.

What You Will Build

  • A REST API
  • CRUD operations
  • JSON responses

Project Structure

project/
 ├── app.js
 ├── routes/
 │    └── items.js
 ├── models/
 │    └── item.js

Setup Project

npm init -y
npm install express

Create Server

const express = require("express");
const app = express();

app.use(express.json());

app.listen(3000, () => {
  console.log("Server running on http://localhost:3000");
});

Create Routes

let items = [];

app.get("/api/items", (req, res) => {
  res.json(items);
});

app.post("/api/items", (req, res) => {
  items.push(req.body);
  res.json({ message: "Item added" });
});

Update Item

app.put("/api/items/:id", (req, res) => {
  items[req.params.id] = req.body;
  res.json({ message: "Item updated" });
});

Delete Item

app.delete("/api/items/:id", (req, res) => {
  items.splice(req.params.id, 1);
  res.json({ message: "Item deleted" });
});

Test Your API

  • Use Postman or browser
  • Send GET, POST, PUT, DELETE requests
