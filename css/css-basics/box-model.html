Box Model

Every HTML element is a box. Understanding how this box works is key to controlling spacing and layout in CSS.

What You Will Learn

  • What the CSS box model is
  • The difference between margin, border, and padding
  • How spacing affects layout

The Box Model Explained

Every element is made up of four parts:

  • Content the actual text or image
  • Padding space inside the box (around content)
  • Border the edge of the box
  • Margin space outside the box

These layers together form the total size of an element.

Example

div {
  margin: 20px;
  padding: 15px;
  border: 2px solid black;
}

This creates space outside (margin), space inside (padding), and a visible border.

Why It Matters

  • Controls spacing between elements
  • Helps align content properly
  • Essential for layout and design

Try It Yourself

  • Create a div with padding and margin
  • Add a border to visualize the box
  • Experiment with different spacing values
