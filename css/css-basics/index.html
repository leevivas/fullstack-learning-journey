CSS Basics — Style Your First Page

Now that you know how to structure a webpage using HTML, it's time to make it look better using CSS.

What is CSS?

CSS (Cascading Style Sheets) is used to control the appearance of HTML elements. It allows you to change colors, fonts, spacing, and layout.

What You Will Learn

  • How to apply CSS to HTML
  • Basic selectors
  • How to change colors and fonts

How to Use CSS

There are three ways to apply CSS:

  • Inline CSS
  • Internal CSS
  • External CSS (recommended)

Example (external CSS):

<link rel="stylesheet" href="style.css">

Basic CSS Syntax

selector {
  property: value;
}

Example:

h1 {
  color: blue;
}

Styling Text

p {
  color: gray;
  font-size: 16px;
}

You can control color, size, and appearance of text easily with CSS.

Try It Yourself

  • Change the color of a heading
  • Change the font size of a paragraph
  • Apply CSS to your HTML page
