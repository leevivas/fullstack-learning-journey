Media Queries

Media queries allow your website to adapt to different screen sizes, making it responsive across mobile, tablet, and desktop devices.

What You Will Learn

  • What responsive design is
  • How to use media queries
  • How to adjust layout for different devices

What is Responsive Design?

Responsive design ensures your website looks good and works properly on all screen sizes—from small mobile phones to large desktops.

Basic Media Query

@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}

This applies styles when the screen width is 600px or smaller.

Common Breakpoints

  • Mobile: up to 600px
  • Tablet: 601px – 1024px
  • Desktop: 1025px and above

Example: Responsive Layout

.container {
  display: flex;
}

@media (max-width: 600px) {
  .container {
    flex-direction: column;
  }
}

On smaller screens, the layout changes from row to column.

Best Practice

  • Design mobile-first
  • Use flexible layouts (Flexbox/Grid)
  • Avoid fixed widths
