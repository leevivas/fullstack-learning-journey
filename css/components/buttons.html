Buttons

Buttons are used for user interaction. Styling them properly improves usability and makes your interface look professional.

What You Will Learn

  • How to style buttons
  • How to use hover effects
  • How to create reusable button styles

Basic Button

button {
  background-color: #2563eb;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
}

Example:

Hover Effect

button:hover {
  background-color: #1d4ed8;
}

This changes the color when the user hovers over the button.

Smooth Transition

button {
  transition: 0.3s;
}

This makes the hover effect smooth.

Button Variations

.primary {
  background: #2563eb;
}

.secondary {
  background: #6b7280;
}

.success {
  background: #16a34a;
}

Example:

Best Practice

  • Use consistent colors and sizes
  • Provide hover feedback
  • Keep buttons clear and readable

  • Add hover effect
  • Create 3 button styles (primary, secondary, success)
