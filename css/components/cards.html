Cards

Cards are reusable UI components used to group related content. They help organize layouts and make your design clean and structured.

What You Will Learn

  • How to build a card layout
  • How to style reusable components
  • How to use cards in real layouts

Basic Card

.card {
  border: 1px solid #e5e7eb;
  border-radius: 8px;
  padding: 15px;
  background: white;
}
