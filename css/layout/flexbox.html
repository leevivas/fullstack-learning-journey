Flexbox

Flexbox is a layout system that allows you to align and distribute elements efficiently. It is one of the most used tools in modern web development.

What You Will Learn

  • How Flexbox works
  • How to align items horizontally and vertically
  • How to create flexible layouts

Basic Concept

To use Flexbox, you apply display: flex to a container. All child elements become flex items. 

.container {
  display: flex;
}

Alignment

.container {
  display: flex;
  justify-content: center;
  align-items: center;
}
  • justify-content — horizontal alignment
  • align-items — vertical alignment

Example Layout

.container {
  display: flex;
  gap: 10px;
}

.box {
  background: #2563eb;
  color: white;
  padding: 20px;
}

This creates a row of boxes aligned in one line.

Common Use Cases

  • Navigation bars
  • Centering content
  • Card layouts
  • Responsive layouts

Best Practice

  • Use Flexbox for 1D layouts (row or column)
  • Combine with media queries for responsiveness
  • Keep layouts simple and readable
