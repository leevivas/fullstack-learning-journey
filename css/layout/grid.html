CSS Grid

CSS Grid is a powerful layout system that allows you to create structured layouts using rows and columns.

What You Will Learn

  • How to create grid layouts
  • How to define rows and columns
  • How to position items in a grid

Basic Concept

To use Grid, apply display: grid to a container. 

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
}

This creates 3 equal columns.

Positioning Items

.item {
  grid-column: span 2;
}

This makes an item span across multiple columns.

Sample Page Layout

.container {
  display: grid;
  grid-template-columns: 1fr 3fr;
}

.sidebar {
  background: #e5e7eb;
}

.main {
  background: #ffffff;
}

This creates a sidebar and main content layout.

Common Use Cases

  • Page layouts (sidebar + content)
  • Dashboard designs
  • Gallery grids

Best Practice

  • Use Grid for 2D layouts
  • Use Flexbox for simple alignment
  • Combine both for powerful designs

Try It Yourself

  • Create a 3-column layout
  • Create a sidebar + content layout
  • Experiment with different grid sizes
